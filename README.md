# Podcast API 🎙️

Uma API funcional desenvolvida para listagem e busca de podcasts, utilizando o método **GET** para o retorno dos dados.

## 🚀 Sobre o Projeto
Este repositório foi desenvolvido como parte prática do curso **Node.js Basics** da **DIO (Digital Innovation One)**.

## 🛠️ Tecnologias
* **100% TypeScript**: Desenvolvimento com tipagem estática para maior segurança.
* **Node.js**: Ambiente de execução para o backend.

## 📖 Funcionalidades
* **GET** `/podcasts`: Retorna a lista completa de programas.
* **GET** `/podcasts?name=...`: Filtra podcasts por nome via query string.
